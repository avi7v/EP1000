Ultimaker Cura

Jun 26, 2020

There are many 3D printer but what we using Ultimaker cura 2+ in our workshop. i started by downloading the cura software from the official website (link) then import file by dragging the Stl file onto the software screen. if you are printing a single model from fusiion, you can just right click the body on the left pannel and click export stl file which is much faster then the normal export. Save the file into a SD card nd insert into the cura 3D printer. press print and select the file and you are good to go.

When i load and slice my nametage onto the cura software, it indicated that it takes about 40 minutes to print but it depends on the quality, speed, support and infill.
